# Roadmap

## 1.0.x

- Improve stability, documentation, and packaging.
- Keep CPU-only defaults and local privacy boundaries.

## 1.1

- Add more configurable performance and threshold options.
- Improve health diagnostics and report summaries.

## 1.2

- Add offline evaluation scripts for public datasets when licensing permits.

## Future Research

- Explore ONNX or native inference as optional research work.
- Separate larger UI modules only when it improves maintainability.

The project does not promise production vehicle deployment.
