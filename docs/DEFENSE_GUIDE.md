# Defense Guide

## 5 分钟讲稿

DriveGuard-AI 是一个基于 Qt、OpenCV 和 MediaPipe 的疲劳驾驶智能预警虚拟仿真系统。它的重点不是单一算法，而是完整软件闭环：视频采集、特征提取、规则融合、事件记录、语音提醒、报告导出和本地隐私保护。

AI 体现在 MediaPipe 面部关键点检测，以及基于视觉特征的可解释规则融合。Qt 负责桌面交互，Python 负责视觉处理，两者通过 QProcess 和 JSON Lines 解耦。

虚拟仿真用于答辩稳定展示，因为真实摄像头容易受光照、隐私开关和权限影响。摄像头和视频模式仍然保留，可以作为真实输入验证。

## 常见问答

- 为什么不需要独显：MediaPipe 和 OpenCV 默认 CPU 运行，项目目标是普通电脑课程演示。
- 准确率如何回答：当前没有公开数据集评测，不虚构准确率，只展示规则和工程闭环。
- Haar 是什么：它是降级链路，只做基础人脸检测和仿真特征补全，不等同于精确关键点。
- 是否真实检测：MediaPipe 模式是真实关键点测量；仿真模式是可复现演示输入。
- 局限是什么：遮挡、侧脸、反光、低光、摄像头角度和性能差异都会影响结果。
