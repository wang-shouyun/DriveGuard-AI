# Privacy

DriveGuard-AI 默认本地运行，不上传云端。

The runtime directory may contain latest frames, event screenshots, SQLite databases, CSV files, and HTML reports. These files can include faces. Use the software only with the consent of people being recorded.

Delete `runtime/` when you need to clear local data. Do not commit runtime data, real driver images, or generated databases to GitHub.

This project does not provide legal, regulatory, vehicle-safety, or medical compliance guarantees.
