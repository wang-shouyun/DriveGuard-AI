# Known Limitations

- Glasses and lens reflection can reduce eye feature quality.
- Low light, backlight, and overexposure can cause invalid frames.
- Side faces, masks, hands, and hair occlusion can break landmark tracking.
- Camera position strongly affects EAR, MAR, and pose estimates.
- Haar fallback uses degraded simulated feature completion.
- PERCLOS is a short-window approximation for demonstration.
- The project has not been validated on large public datasets.
- CPU performance differs across ordinary Windows computers.
- This is not a production vehicle safety system.
