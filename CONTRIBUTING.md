# Contributing

Thank you for improving DriveGuard-AI.

- Use Python 3.9 and Qt 6.
- Run `python -m unittest discover -s tests -v` and `python scripts/smoke_test.py` before opening a pull request.
- Do not commit real driver face images, runtime databases, screenshots, or local absolute paths.
- Do not claim production vehicle safety, medical diagnosis, or unmeasured accuracy.
- Explain privacy impact when changing capture, storage, report, or logging behavior.
- Update README, DESIGN, protocol docs, or tests when behavior changes.
- Pull requests should include the commands run and their results.
